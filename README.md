# Examen Project Euromoon

Een Java-console applicatie om reizen en bijhorende objecten te beheren.
Deze applicatie werd ontwikkeld met Java 25.
Dit werd ontwikkeld in opdracht van een examen programmeren in Java.

## Beschrijving

Deze applicatie laat toe om verschillende trein-gerelateerde objecten aan te maken en te beheren, zoals het maken van reizen met een aankomstpunt en een eindpunt, of een aantal passagieren te registreren voor een reis.

## Vereisten
- Java 
- IDE om de applicatie te laten werken

## Project starten
1. Open het project in een IDE die Java accepteert  
2. Ga naar `src/Main.java`  
3. Run de `main`-methode
4. Checken dat een bestand "ticket" bestaat buiten de src bestand om het wegschrijven van een boardinglijst mogelijk te maken
